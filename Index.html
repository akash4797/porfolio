ARC & CIV ProjectMan

Software Development, Portfolio

Introducing a JAVA-based software using SQLITE for database management.

Main Feature:
This will be used for those people who want to record their working projects' data and to see the progression of their projects. Moreover, it will help them to put their drafts (docs, excel etc) or project works in the software and it will make a backup and automatically shows up the progression. Also, it will help them to manage their currency (like Credit, Debit etc). This software can be used in offices as this can be put in servers so that employers can use them.

Other Features:
First of all, this software is based on multiple user accounts including an admin user that has administrative control. He can generate reference keys to add new users and besides he can delete also. In Addition, he can use the currency management processes. Secondly, Users can add their own information (Name, address, image etc) and can use the project works. Lastly, as this can be used in the server so that everyone can back up their working files easily. Moreover, they don't have to show their work progression to their boss. He can check the progression of this software.

Still working and developing this with some new features. Hope it will be done within a week only and can be used for clients.

NB: This is just a demo by adding some temporary names. Even the software's name will be changed.

PERSONAL COMMAND PROMPT

Software Development, Portfolio

A simple command prompt for perosnal usage. It can be used for creating own language or command to do various custom works. It has folder hidding feature to create secret folder. Also, It can be used to store personal data. An exclusive feature of this app is train. This will help users to train the app for understanding and creating own language in a shortcut way. It is a simple tool or widget for personal use.

This app had designed by java swing with very basic knowledge but the concept is very clear to the users. Here, bufferedReader fuction has implemented to read the stored language or command package from a text file only. Users can change their commands or train the app by changing the text file.

This is an open source app so that people who want to learn or develop software can use them.
